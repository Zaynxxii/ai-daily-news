# AI Daily News 🤖

每日 AI 新闻精选网站，自动生成并部署到 GitHub Pages。

## 🌐 访问

https://Zaynxxii.github.io/ai-daily-news/

## 📁 文件说明

- `index.html` - 生成的网站主页
- `template.html` - 网站模板
- `generate-site.sh` - 网站生成脚本
- `deploy.sh` - 部署脚本

## 🔄 更新流程

### 手动更新

```bash
# 1. 生成新闻
cd /root/.openclaw/workspace/ai-daily-digest
./send-digest.sh

# 2. 生成网站
cd /root/.openclaw/workspace/ai-daily-news
./generate-site.sh

# 3. 推送
git add -A
git commit -m "更新新闻"
git push
```

### 自动更新（需配置）

1. 创建 GitHub Token（需要 `workflow` 权限）
2. 在 GitHub Settings → Secrets 添加 `GITHUB_TOKEN`
3. GitHub Actions 会自动部署
